WishList

WishLIst is an app where you can a User can login and add items to his wish list, see items on other users list and add their items onto there own list, and be able to click on an item and see who has it on there list.
I structured the database and implemented the front end design plus deploying the app using Amazon AWS (EC2) all within the 4.5 hours alotted test time.
Developed using Django, Python, HTML, CSS/bootstrap, and SQlite.
